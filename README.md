# 🎬 CineAI-X

> An intelligent movie discovery engine powered by hybrid AI — blending content, behavior, and mood into one seamless experience.

---

## 🧠 Overview

CineAI-X is not just a recommender system — it's a personalized cinematic intelligence layer.

It analyzes:
- 🎭 Content similarity (NLP + TF-IDF)
- 📊 Popularity & rating signals
- 🕒 Recency dynamics
- 🎯 Diversity optimization
- 😶‍🌫️ Mood-based intent

…to generate high-quality, human-like recommendations.

---

## ⚡ Features

### 🔍 Smart Discovery Engine
- Fuzzy search with intelligent matching
- Hybrid scoring system (5-signal weighted model)
- Real-time ranking with explainability

### 😶‍🌫️ Mood-Based Recommendations
- Emotion-driven filtering (Happy, Dark, Mind-Blown…)
- Keyword + genre semantic mapping

### 👤 User Taste Profiling
- Watch history tracking
- Like/skip feedback learning
- Dynamic genre preference modeling

### 📊 Analytics Dashboard
- Rating distribution insights
- Genre trends
- Popularity vs quality analysis

### 🎨 Premium UI
- Custom cyber-green theme
- Smooth card-based layout
- Real-time responsiveness
